Window Functions in SQL

Posted at — Apr 1, 2020 DRAFT

SQL is an important tool in data science. The first step in Model Building involves understanding the data, where it’s stored and how to bring it together. This usually involves pulling data stored across many tables in multiple schemas: - Joining them with the help of primary keys - Watching out for ‘one to many’ joins - Figuring out the tests that have happened in the past (so we can understand which is the data that’s to be included and which shouldn’t be)
